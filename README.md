ember-casl
==============================================================================

This package allows to integrate `@casl/ability` with [Ember](https://emberjs.com/) application.


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.20 or above
* Ember CLI v3.20 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-casl
```
